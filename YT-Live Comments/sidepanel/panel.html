One-time setup (macOS):

  1. Install yt-dlp via Homebrew (skip if already installed): 
    brew install yt-dlp
  2. Register the native messaging host so Chrome can launch yt-dlp on this extension's behalf. From the extension folder: 
    bash install_host.sh
    When prompted, paste your extension ID from chrome://extensions (enable Developer mode to see it).
  3. Reload this extension at chrome://extensions, then click Re-check below.

Don't have Homebrew? Install it from brew.sh first. Other archive features (metadata, comments, live chat, screenshots) work without yt-dlp — only the video file download requires it.

Note: the installer copies the host script into ~/Library/Application Support/com.ytarchiver.downloader/. macOS Sonoma and later block Chrome's native-host child processes from reading files in ~/Documents, ~/Desktop, and ~/Downloads (TCC), so the host can't live there. Logs are at ~/Library/Logs/com.ytarchiver.downloader/ if you ever need to debug.
