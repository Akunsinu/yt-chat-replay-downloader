One-time setup (macOS):

  1. Install yt-dlp via Homebrew (skip if already installed): 
    brew install yt-dlp
  2. Register the native messaging host so Chrome can launch yt-dlp on this extension's behalf. From the extension folder: 
    bash install_host.sh
    When prompted, paste your extension ID from chrome://extensions (enable Developer mode to see it).
  3. Reload this extension at chrome://extensions, then click Re-check below.

Navigate to a YouTube video to get started.

Setup walkthrough

Step 1 of 5

Install Homebrew

Already have Homebrew? Skip ahead — click the next dot or hit Next →.

  1. Open Terminal (Cmd+Space → "Terminal" → Enter).
  2. Copy the command below, paste it into Terminal, and hit Enter. It will ask for your Mac password partway through — that's normal.
Terminal
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
==> Checking for `sudo` access (which may request your password)...
Password: ●●●●●●●●
==> Downloading and installing Homebrew...
==> Installation successful!
$ ▍
Step 2 of 5

Install yt-dlp

  1. In Finder, locate the YT-Live Comments folder.
  2. Right-click it → New Terminal at Folder.
  3. Run the command below in that Terminal window.
Terminal — YT-Live Comments
$ brew install yt-dlp
==> Downloading yt-dlp...
==> Pouring yt-dlp...
🍺 /opt/homebrew/Cellar/yt-dlp/2026.03.17: 23 files
$ ▍
Step 3 of 5

Find your extension ID

Chrome assigns each extension a unique ID. The installer needs it so it can grant only this extension permission to talk to yt-dlp.

chrome://extensions
Extensions
Developer mode
① Turn this on
▶
YouTube Video Archiver
Archive YouTube videos with metadata, comments, live chat…
ID: kofmgaibpgeijecaiplbeggaifmkcdep
② Copy this whole string

Open chrome://extensions in a new tab. Toggle Developer mode on (top-right). The ID is 32 lowercase letters under "YouTube Video Archiver." Select and copy it.

Step 4 of 5
